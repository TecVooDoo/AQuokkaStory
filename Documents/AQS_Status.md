**Major design decisions this session:**
- Joeys are **autonomous companion AI**, not inventory items (Pikmin-like)
- Follow line is **emergent** based on proximity -- no managed list
- Pouching requires **physical proximity** to Mom
- Joeys **stop at ledges** they can't cross alone (future: 3+ Joeys form ladder)
- Joeys **collide with Mom and each other** -- pile-ups are intentional humor
- Joeys are **indestructible** (except Normal Joey) but can be **kidnapped by predators**
- Mom-dodge direction is **+X (toward camera)** -- deferred to polish pass
- 2.5D axes: **Z = lateral movement, X = depth (locked), Y = up**

## Development Plan

### Sprint 1: Core Feel
- Install Tier 1 packages (defaults + confirmed AQS stack)
- Import Scorch from HOK as placeholder character
- Prototype Mom movement: hop-based locomotion (fast hop, slow hop), bipedal stance, climb with stamina
- Gray-box test level with platforms, walls, climbable surfaces
- Get platforming feel right before adding Joey mechanics

### Sprint 2: Joey Launch MVP
- Implement aim + trajectory arc + launch + recall
- One Joey (Lead) with bowling ball ability
- Basic hit detection and feedback (Feel + Damage Numbers Pro)
- RayFire for breakable barriers (Lead Joey smashes through)
- Trajectory display: Toolkit for Ballistics 2026 (ENTRY-208) + ALINE

### Sprint 3: Joey Variety
- Add 2-3 more Joeys (Ballet freeze, Helium inflate, Ninja shuriken)
- Inflate mechanic prototype (start with DOTween scale tween, eval MegaFiers 2 ENTRY-136 or Deform ENTRY-153 if needed)
- Pouch management (swap active Joey, equipped Joey modifies Mom stats)
- Ultimate Selector (ENTRY-029) for radial Joey selection wheel (replaces blind Q/E when 3+ Joeys)
- Rope Toolkit (ENTRY-271) for vine swinging traversal

### Sprint 4: Enemies
- 3 enemy types with basic AI covering distinct archetypes (stealth, pack, ambush)
- Joey-specific weaknesses working
- **Primary enemy models: Poly Art Forest Set (ENTRY-336)** -- AC-native, drops into existing Malbers setup with zero rewiring. Supersedes Polyperfect placeholders for this sprint.
  - **Fox** -- stealth/solo Joey-kidnapper. Snatches the Normal Joey (the one destructible Joey), flees to a den. Core Joey-kidnap mechanic proof.
  - **Wolf** -- pack predator (2-3 spawn together). Chases Mom, forces territorial retreat. Tests chase AI + group behavior.
  - **Cougar** -- ambush predator. Perches on ledges/branches, drops on Mom from above. Tests LOS triggers + vertical AI.
  - **Held for later:** Bear (tank miniboss), Boar (charge attack), Tiger (boss), Moose (neutral), Deer/Rabbit (friendly/ambient).
- Polyperfect animals remain as fallback for species the Forest Set lacks (e.g. snake, hawk, otter -- ambient swamp fauna).
- AI stack: SensorToolkit2 (ENTRY-231) + Behavior Designer Pro (ENTRY-229) + A* Pathfinding Pro (ENTRY-164)
- Ragdoll Animator 2 (ENTRY-216) for enemy death reactions

### Sprint 5: Vertical Slice
- One playable level (Tutorial or Swamp)
- All core systems integrated
- Malbers Quest Forge (ENTRY-297) for quest tracking, minimap/world map reveal, dialogue, compass
- AA Map and Minimap evaluation (may be superseded by Quest Forge's built-in minimap)
